# Sobre Mim:
Sou Desenvolvedor Back-end com atuação em Full Stack, focado no desenvolvimento de APIs RESTful utilizando Node.js, integradas a bancos de dados relacionais e não relacionais. Possuo experiência no suporte à camada de front-end com React e Next.js, o que me permite compreender o fluxo completo das aplicações web.<br><br>Tenho atenção especial à organização de código, boas práticas, separação de responsabilidades e performance, buscando sempre soluções simples, escaláveis e fáceis de manter. Demonstro interesse contínuo por arquiteturas modernas, segurança e qualidade técnica no desenvolvimento de software. Sou um profissional disciplinado, responsável e com comunicação clara, com facilidade para aprender rapidamente e evoluir em ambientes colaborativos.
